# 基础组件模块
### 打包操作说明：
```
npm run pub
```
将会自动将src目录下的fonts、style、util打包到lib目录下，可以公共使用

### 使用组件方式

## Sept-1:
import {Button[,Input]} from 'component-base-ui'

## Sept-2:
Vue.component(Home.name, Home);
...

### See the demo

## Sept-1:
npm install --registry=http://192.168.1.142:4873

## Sept-2:
npm run dev
